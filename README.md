# README

> rime-pinyin-normal 是我个人使用和维护的一个中州韵输入法操作工具库。

目录结构

```bash
[root@fedora ~]# tree ~/rime-pinyin-normal/
~/rime-pinyin-normal/
├── android_pinyin_ime    # 安卓IME默认的词库转换rime词库工具
├── merge_userdb          # 合并rime产生的词典快照工具
├── sogou_dict_generate   # 搜狗词库生成工具
└── README.md             # 使用说明
```

每个目录下都有对应的说明用法。
